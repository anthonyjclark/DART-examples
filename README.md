# DART-examples

A few simple DART examples.

Build with:

```bash
./build build <example-name>
```

Run with

```bash
./build run <example-name>
```

The build script adds `$HOME/.local` to the inlcude path and to the `DYLD_LIBRARY_PATH` variable.
